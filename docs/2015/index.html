2015年度証明論研究集会
SLACS 2015 - 記号論理と情報科学研究集会

2015年度の「証明論研究集会」と「記号論理と情報科学研究集会（SLACS）」を合同開催します．どちらの研究集会も数理論理学，計算の理論，情報科学などの理論・応用に関係する様々な話題が発表されていて研究者・学生の貴重な相互交流の場となっています．完成された研究のみならず発展途上の話題も大歓迎ですので，気軽に講演・参加を御検討下さい．

開催日：2015年12月14日（月）〜16日（水）

プログラム
学生で講演を申し込まれた方の中で旅費の援助を希望される方がいましたら遠慮なくお申し付け下さい．

場所：南山大学名古屋キャンパスS棟（教室棟）7階S74教室　（S棟には教室棟(7階建)と研究棟(4階建)がありますのでご注意ください．）
アクセス マップ　（八事日赤駅の他に名古屋大学前駅からもアクセス可能です．）

懇親会：15日の夜を予定していましたが，都合により14日に変更することとなりました．大変申し訳ありません．
　　　　予算3500円で，会場近傍のお店となりました．
