SLACS 2014 / 2014年度超準解析シンポジウム

2014年度のSLACS及び超準解析シンポジウムを、以下の通り一部合同で開催いたします。
どちらの集会も、例年とは時期が異なりますのでご注意ください。

日時：5月26日（月） ～ 5月28日（水）

但し、原則として、SLACSの講演は26,27日、超準解析シンポジウムの講演は27,28日とし、
27日は一部交流形式を取る予定です。

場所：東京工業大学大岡山キャンパス
           西８号館Ｅ棟10階 大会議室

この地図の25番という建物です。 大岡山キャンパスまでの交通案内はこちらです。


懇親会: 5月27日（火）18:00-

会場: 鴻元食坊

会費: 学生3000円、一般4000円程度を予定


wSLACS枠は講演を締め切りました。
超準解析枠は引き続き受付いたします。
講演募集を締切りました。

＞講演のお申し込み、参加登録はこちらのフォームをお使いください。



Information

  • 2つの研究集会の交流の一環として、次の二つの基調講演を予定しております。
    ・蓮尾一郎先生（東京大学、SLACSより）
    ・小澤正直先生（名古屋大学、超準解析シンポジウムより）
  • 特別講演として、東京大学名誉教授の斉藤正彦先生より、「超準考古学」というタイトルでお話をいただける予定です。

Organizer: 横山 啓太, Local organizer: 鹿島 亮
