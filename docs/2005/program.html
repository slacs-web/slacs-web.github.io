講演概要

田村 晋照（東京工業大学 情報理工学研究科 数理・計算科学専攻） difference operater を含む時制論理について
時制論理にdifference operater D を加えた様相論理に関してtree sequentによる公理化を行い、完全性、及びsahlqvist completeness theorem （sahlqvist formula と対応する一階述語論理式をみたすフレームのクラスに関する強完全性）を示す。

松本 利雅（北陸先端科学技術大学院大学 情報科学研究科 安心電子社会研究センター） 許容可能なカット規則をもちいた様相論理 S4 の反例構成
本講演では, 許容可能なカットという形でカット規則を持つ S4 のシーケント体系にもとづいて, 失敗した証明図から反例を構成する方法を示す. 許容可能なカット規則をもちいて反例を構成すると, カット規則を除いた S4 のシーケント体系にもとづいて構成した反例よりも小さいサイズの反例を得ることができる.

黄 隆裕（埼玉大学大学院理工学研究科情報システム工学専攻） 分散メモリ型並列計算機における前向き演繹エンジンの効率化
近年、自動定理発見システムや先行予測システムなど様々な情報システムにおいて、与えられた前提から新しい結論を導出できる前向き演繹エンジンが求められている。 実用的な前向き演繹エンジンを実現する際の最大の課題は効率である。 なぜなら、前向き演繹エンジンは必要とする結論を導出するためには前向き演繹を繰り返し行わなければならない。 そのため、膨大な数の結論を処理しなければならず、本質的に非効率な処理となってしまうためである。 そのため、並列計算による前向き演繹エンジンの効率化が試みられている。 その計算モデルの一つとして2003年、前提分散モデルが提案された。 前提分散モデルでは、前提を分散配置している。 そのため、導出した結論が前提や以前に導出した結論と重複していないかを調べる重複検査処理において、通信を行う必要が生じる。 前提分散モデルで実行時間はある程度短縮されたものの、予測よりも速度向上率が鈍るという問題が発生した。 本講演では、この問題の原因を明らかにし、その解決方法を示す。 また、その方法の有効性を明らかにする。

高橋 勲男（埼玉大学大学院情報システム工学専攻） 認識的プログラミングとその実現のための認識的プログラミング言語
認識的プログラミングとは，計算的方法論という第３の科学的方法論を樹立するために提案された新しいプログラミングパラダイムである． それは，人間の認識過程を強相関論理に基づいてモデル化し，必然的な帰結関係を計算の基本対象とし，認識過程をプログラミングの対象とするものである． 認識的プログラミングを実現するために認識的プログラミング言語を新たに設計しその処理系を実現しなければならない． 本講演では，まず認識的プログラミングが従来のプログラミングパラダイムとどう異なるかについて，認識的プログラミングの概念を説明する． そして，我々がまとめた認識的プログラミング言語に対する要求を述べ，それに基づいた認識的プログラミング言語設計案を説明する．

佐々木 克巳（南山大学数理情報学部） S4の証明可能性を用いずにS4の非同値論理式を構成する方法
SLACS2004において、1変数でしかも⊥が現れない論理式の集合Sに対して、順序集合(S/\equiv,\leq)を帰納的に構成する方法が話者によって述べられた。 ただし、\equivはS4における同値性、\leqはS4における導出関係である。 その方法は、順序集合がブール代数であることを利用し、本質的にはそのブール代数の生成元の代表元(非同値論理式)だけを構成するもので、その中でS4の証明可能性はかなり限定された形の論理式だけに用いている。 本報告では、この結果をn変数でしかも⊥も対象とした論理式へ拡張し、さらに、上の方法で用いられていたS4での証明可能性を、論理式の形のみから判断する方法を述べる。

中澤 巧爾（京都大学大学院情報学研究科），池田 聡（京都大学大学院情報学研究科） SN-proofs by CPS-translations
CPS変換は, Parigotの\lambda\mu計算をはじめとする様々な計算体系の強正規化性(SN)の証明の道具としても利用されている. しかし, それらの証明の多くが, 同じ原因による誤りを含んでいることが指摘され, \lambda\mu計算に対してその修正がなされている(中澤, 龍田, JSL, 2003). さらに, 我々はPPL2005において, より多くの体系に対して適用可能な一般的な修正方法を提案した. 本発表では, この証明方法を紹介し, さらにこの方法が, permutative conversionを含む自然演繹古典論理や, Ong-Stewartや藤田によって提案された値呼び\lambda\mu 計算のSNの証明に適用できることを示す.

角谷 良彦（東京大学） 古典様相論理のための計算系
直観主義様相論理に Curry-Howard 対応する計算系が知られているが，それを拡張して，古典様相論理に対応する計算系を考察する。

岸田 功平（ピッツバーグ大学），佐藤 憲太郎（ミシガン大学） 連続性と論理
連続写像に沿って論理がどのように翻訳されるかを考察する。

竹内 泉（産業技術総合研究所システム検証研究センター算譜意味論研究チーム） ピゾー数による小数展開と色塗り
ピゾー数による非整数底の小数展開が複素平面を埋め尽くすこと、及びこの事実とオートマトンとの関係が知られている。 この既知の結果を紹介すると共に、これを利用したフラクタル色塗りを見せる。

鴨 浩靖（奈良女子大学理学部） Urysohnの万有距離空間の計算可能性について
Urysohnの万有距離空間に計算可能性構造を入れた計算可能距離空間を構成した。 この空間には、すべての可分距離空間から等長な埋め込みが存在するだけでなく、すべての計算可能距離空間から計算可能かつ等長な埋め込みが存在する。 また、Urysohnの万有距離空間の一意性にあたることが、計算可能性も含めて成り立つ。

鹿島 亮（東京工業大学 情報理工学研究科） 直観主義命題論理周辺の二つの問題
最近考えている二つの問題を紹介する。 （１）中間命題論理の公理は古典論理において非一般的であるか？ （２）命題変数への全称量化と含意だけの直観主義論理で、 「エルブランの定理」と似たようなことが成り立つか？

関 隆宏（九州大学 大学評価情報室） A note on metavaluation and γ for relevant modal logics
適切様相論理における metavaluation にはさまざまな定義を与えることが考えられる。 本講演では、可能世界の関係を考慮しない1種類の metavaluation に基づいて metavaluation を定義し、その場合の metacompleteness とγ許容性について論じる。

志村 立矢（日本大学理工学部数学科） Gabbay-de Jongh の logic の disjunction property について
有限 n 分木全体で特徴付けられる Gabbay-de Jongh の logic が disjunction property を持つことはよく知られているが、本講演ではその証明の単純化と精密化について述べ、その系としてより強い性質を持つことを指摘する。 補足として、他の logic に対する同様の考察の展望にも触れる。

三浦 潤一（埼玉大学大学院理工学研究科情報システム工学専攻） 暗号プロトコル形式分析の推論的手法
暗号プロトコルの安全性を理論的に分析する従来の手法は殆ど，証明的・検証的手法である． しかし，証明的・検証的手法は，証明・検証したい性質を予め明確に定義した分析対象しか適用できないため，分析者の想定していない未知の欠陥や性質を発見することが一般的にはできない． 暗号プロトコルにおける欠陥や性質を発見する系統的な方法論を樹立するために，我々は，推論的手法という新しい形式分析手法を提案し，その実現に向けた課題を明らかにする． 暗号プロトコル形式分析の推論的手法とは，形式化された暗号プロトコルに関する一般的な原理や性質および対象暗号プロトコルについて既知の事柄を前提として，規範（義務）相関論理に基づく推論を行なうことで，分析者の知らなかった，未知の性質や欠陥を発見する事を目的とした手法である． 本講演では，推論的手法の概念とその実現に向けて解決しなければならない課題，および研究の現状とこれからの展開について述べる．

奈良 信介（埼玉大学大学院理工学研究科情報数理科学専攻） 様相演算子を含む論理体系に基づいた前向き演繹エンジン
現在、法的推論エンジン、地理情報システムや先行予測反応システムなど様々な分野の発見や予測を目的とした知識ベースシステムに置いて、様相演算子を含む論理体系に基づいた前向き演繹エンジンの開発が求められている。 しかし、現在のところ、様相演算子を含む論理体系に基づいた前向き演繹エンジンは実現されていない。 本発表では、私たちの開発した様相演算子を含む論理体系に基づいた前向き演繹エンジンについて述べる。 まず、様相演算子を含む論理体系に基づいた前向き演繹エンジンの要求と機能を明らかにする。 そして、今回の実装における課題と今回行なった実験について述べる。 その後、私たちの実装した様相論理に基づいた前向き演繹エンジンを利用する事により実現可能な応用システムについて述べる。 最後に今後の課題について述べる。

佐野 勝彦（京都大学文学研究科），鈴木 徹（東京工業大学 理学部 情報科学科） Predicate Extension of Bimodal Logics with Irreflexive Modality
「『現在の世界から到達可能で、かつ現在と異なるすべての可能世界』において真」を表す様相演算子をもつ様相論理の述語拡大を考える. 本発表では, この述語拡大が, 拡大変域フレームのクラスに対して完全性をもつか, という問題を扱う. まず, 対応する命題様相論理の場合に使えた論法が適用できない, という述語様相論理に固有の難点があることを指摘する. 次に, その難点を回避して完全性を証明できることを示し, 最後に, その回避法と, tree sequent の発想の類似点を述べる.
