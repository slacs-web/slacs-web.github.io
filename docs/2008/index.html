桜島

開催期間と場所は以下の通りです。

  • 開始予定：2008年8月27日（水） 午後
  • 終了予定時刻：2008年8月29日（金） 午前または午後
  • 開催場所：鹿児島大学理学部 1号館101（アクセス情報， 館内配置図）

広くご講演を募集いたします．皆様お忙しい時期だとは思いますが，ご検討頂ければ幸いです．詳しい情報は追ってお知らせします． 募集は終了しました．

最終日29日の午後まで開催するかどうかは，講演の数によります． は講演ありません．

講演される方は，slacs-algi2008@m.aist.go.jp まで情報をお寄せください．題目，講演者名とe-mail address，所属，梗概，希望時間，講演に必要な道具などが記されていればありがたいですが，事情によりスロットが不足する可能性もございますので「講演したいと思っている」程度の不完全情報でも結構です.

時間の空きがある限り，開催直前まで講演申込を受け付けるつもりですが，「講演する意志がある」方はどうぞ，お早めに御一報ください．

皆様のご参加をお待ちしています． 36名の皆さんにご参加頂きました．ありがとうございました．

宿泊

鹿児島中央駅や天文館周辺はホテルがたくさんありますし，大学へのアクセスも便利です．

疲れがたまっている方

銭湯をお勧めします．「330円で天然温泉に入れます」だそうなので．詳しくは鹿児島県公衆浴場業生活衛生同業組合のＨＰをご覧下さい．

鹿児島の日差し

鹿児島はやっぱり南国なんですね．日差しの強さは恐らく皆さんの想像以上だと思います．帽子やサングラス，日傘などを準備されることをお勧めします．
また，暑さに負けてしまいそうなときは冷たーいしろくまがお勧めです．
